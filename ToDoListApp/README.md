# To-Do List App

## Overview
This project is a simple Android To-Do List application designed to demonstrate modern Android development practices. Users can add, view, and manage tasks through a clean and intuitive interface.

## Technologies Used
- Kotlin
- Android Studio
- Jetpack components
- MVVM architecture (planned)
- Room Database (planned)

## Purpose
This project was created as part of coursework to practice Android application structure, state management, and data persistence concepts.
