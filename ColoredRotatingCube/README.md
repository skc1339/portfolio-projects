# Colored Rotating Cube

## Overview
This project demonstrates a 3D colored cube rendered using WebGL. Each face of the cube has a unique color, and the cube rotates continuously along the X and Y axes.

## Features
- WebGL-based 3D rendering
- Vertex and fragment shaders
- Depth testing for correct face overlap
- Smooth animation using requestAnimationFrame

## Technologies Used
- JavaScript
- WebGL
- HTML5 Canvas
- glMatrix

## How to Run
Open `index.html` in a modern web browser that supports WebGL.
